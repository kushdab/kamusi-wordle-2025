# Kamusi Wordle 2025

A Swahili-language version of the popular word-guessing game. 

## Jinsi ya Kucheza (How to Play)
1. Fungua `index.html` kwenye kivinjari chako (Open index.html in your browser).
2. Unashinda kwa kukisia neno la herufi 5 la Kiswahili.
3. Rangi ya kijani inamaanisha herufi ipo sehemu sahihi.
4. Rangi ya njano inamaanisha herufi ipo lakini sehemu tofauti.
5. Rangi ya kijivu inamaanisha herufi haipo kabisa.

## Requirements
- Any modern web browser.
